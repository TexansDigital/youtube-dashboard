Texans YouTube Dashboard

Last updated: January 10, 2026

Introduction

Texans YouTube Dashboard ("the Application") is an internal analytics tool developed for and used exclusively by the Houston Texans digital marketing team. This privacy policy explains how the Application accesses, uses, and protects data from the YouTube Analytics API.

Application Purpose

The Application is designed to help the Houston Texans digital marketing team:

  • Monitor YouTube channel performance metrics (views, watch time, subscriber growth)
  • Identify high-performing video content and optimization opportunities
  • Analyze audience retention patterns to identify potential YouTube Shorts clips
  • Compare performance across different time periods
  • Generate SEO recommendations for video titles and descriptions

Data We Access

The Application accesses the following data from the YouTube Analytics API for the Houston Texans YouTube channel only:

  • Channel statistics (subscriber count, total views, video count)
  • Video performance metrics (views, watch time, likes, comments, shares)
  • Audience retention data (average view duration, percentage watched)
  • Traffic source information (where viewers discover videos)
  • Video metadata (titles, descriptions, publish dates)

How We Use Data

All data accessed through the YouTube Analytics API is used exclusively for:

  • Displaying analytics dashboards to authorized team members
  • Generating automated reports on channel performance
  • Identifying content optimization opportunities
  • Internal decision-making regarding content strategy

Data Storage

Analytics data is cached in JSON files hosted on GitHub Pages to enable dashboard functionality. This data is refreshed hourly via automated processes. No personal user data is collected or stored.

Data Sharing

We do not sell, trade, or share YouTube Analytics data with any third parties. Data accessed through this Application is used solely for internal Houston Texans operations.

Data Security

Access to the Application and its underlying systems is restricted to authorized Houston Texans digital marketing team members. OAuth credentials are stored securely using GitHub Secrets encryption.

Third-Party Services

This Application uses the following third-party services:

  • Google YouTube Analytics API - To access channel performance data
  • Google YouTube Data API - To access video metadata
  • GitHub Pages - To host the dashboard application
  • GitHub Actions - To automate data refresh processes

User Rights

This Application accesses only the Houston Texans brand YouTube channel data and does not collect any personal information from end users or website visitors.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.

Contact Information

For questions about this privacy policy or the Application's data practices, please contact the Houston Texans digital marketing team.
